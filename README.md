# Task Management Web App

## Overview

Task Management Web App is a Django-based application that helps users organize and manage their daily tasks efficiently.

## Features

- User Registration and Login
- Add New Tasks
- Update Existing Tasks
- Delete Tasks
- Mark Tasks as Completed
- User-Friendly Dashboard

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Project Structure

manage.py
templates/
static/
db.sqlite3
